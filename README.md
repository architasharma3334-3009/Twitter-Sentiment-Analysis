# Twitter Sentiment Analysis

## 📌 Project Overview
This project performs sentiment analysis on Twitter data using Natural Language Processing (NLP) and Machine Learning techniques. The goal is to classify tweets as positive or negative based on their content.

## ⚙️ Technologies Used
- Python
- NLP (Natural Language Processing)
- Scikit-learn
- NLTK
- Pandas, NumPy
- Matplotlib, Seaborn

## 🔍 Features
- Data preprocessing and cleaning
- Text normalization and tokenization
- TF-IDF feature extraction
- Multiple ML models:
  - Naive Bayes
  - Logistic Regression
  - Support Vector Machine (SVM)
  - Random Forest
  - AdaBoost
- Model evaluation and comparison

## 📊 Results
- Compared multiple models based on accuracy
- Best performing model: Support Vector Machine (SVM)

## 📁 Files in this Repository
- Jupyter Notebook (Sentiment Analysis)
- Data Visualization Notebook
- Research Report

## 🚀 Conclusion
This project demonstrates how NLP and machine learning can be used to analyze social media data and classify sentiments effectively.
